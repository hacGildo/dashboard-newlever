Como Salvar e Usar Este Dashboard

1. Clique com o botão direito em qualquer área da página e selecione "Salvar como"

3. Salve o arquivo com um nome como "dashboard-newlever.html"
4. Abra o arquivo salvo em qualquer navegador (Chrome, Firefox, Edge)
5. Os gráficos são interativos - passe o mouse para ver detalhes
